This is a sourcing reference, not a guaranteed-current catalog. Sizes, prices, and availability shift over time — verify on the supplier site before importing. Use the **Import from Amazon** flow (Admin → Products → Import) to pull each one in: paste the URL + image URL + page text, review the extracted draft, save.

## Suggested import order

If you're spinning up the catalog from scratch, this order tends to load nicely on the homepage filter chips:

1. Start with **Supplements** — biggest category, broadest appeal, easiest to merchandise.
2. Add **Herbal teas** next — small SKU count but high give-as-gift potential, fills out the "all natural" vibe.
3. Then **Essential oils** — Plant Therapy alone gives you 5 products with consistent quality.
4. **Nutrition** and **Skincare** can grow more slowly as you find brands you trust.
5. **Fitness** is the smallest category by intent — keep it tight, NSF-tested only.
